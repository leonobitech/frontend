## TL;DR

When Claude gets it wrong, it's usually a briefing problem — not a capability issue. Most challenges have simple fixes: add context, be explicit about format, show examples, and describe the tone you want.

**Key highlights from this episode:**

- Common fixes: too generic (add context), too long (set word limits), wrong format (show examples), wrong tone (describe it).
- Treat your first prompt as a starting point — iterate through specific feedback.
- The 4D Framework: Delegation, Description, Discernment, Diligence — the core competencies of AI Fluency.
- Run simple evals: gather examples, test prompts, compare, refine.
