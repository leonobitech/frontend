## TL;DR

The Claude desktop app gives you three distinct ways to work: **Chat** for quick exchanges and brainstorming, **Cowork** for complex research and finished deliverables, and **Code** for building software with a full dev environment.

**Key highlights from this episode:**

- Chat: quick entry (double-tap Option), screenshots, dictation, desktop connectors.
- Cowork: multi-source research, scheduled tasks, browser use, plugins — work that takes real effort.
- Code: Ask/Code/Plan modes, visual diffs, git integration, local and remote environments.
- Cowork and Code both run on Claude Code underneath — capable of independent, long-running tasks.

This is Episode 04 of **Knowing Claude over Coffee** — a series breaking down Claude's ecosystem one concept at a time.
